Connection Troubleshooting
Common Issues:
  • Timeout: Network connectivity issue or wrong IP
  • Connection refused: API service not enabled on router
  • Authentication failed: Wrong username/password or username format issue
Quick Checks:
  1. Verify router IP address is correct
  2. Ensure API service is enabled on router
  3. Check user has API permissions
  4. Test network connectivity (ping router IP)
  5. Verify port 8728 is accessible
Router Configuration:
# Enable API service
/ip service set api disabled=no

# Create API user (if needed)
/user add name=api-user password=your-password group=full
